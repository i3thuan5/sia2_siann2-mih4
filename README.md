[寫啥物](http://xn--7zrr5mu7u.xn--v0qr21b.xn--kpry57d)提供一般使用者臺語文字正規化和語言朗讀。


## 開發
### 安裝
1. 先裝`nodejs 6`
2. 裝相關套件`npm i`
3. `npm start`，即可看`http://localhost:3000/`

### 排版
本專案用`jscs`排版，可下指令`npm run reformat`自動排版

### 佈署
下指令`npm run deploy`
